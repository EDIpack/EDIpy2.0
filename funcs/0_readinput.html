Input file manipulation

edipack2py.global_env.read_input(input_string)[source]

This function reads from the input file of EDIpack. If the file does not exist, a template file is generated with default parameters. This is generated with the prefix “used.” which will need to be removed for it to be read. “used.${input_string}” will be updated within the DMFT loop with the current value of the input variables.

Parameters:

input_string (str) – The name of the input file to be read, including the extension

Returns:

Nothing

Return type:

None
